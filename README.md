# Stock Price Prediction using LSTM

PROJECT OVERVIEW

This project predicts Apple stock closing prices using a Long Short-Term Memory (LSTM) neural network.

TECHNOLOGIES USED

* Python
* NumPy
* Pandas
* TensorFlow
* Scikit-Learn
* Matplotlib

DATASET

Apple historical stock market data.

FEATURES

* Data preprocessing using MinMaxScaler
* LSTM model training
* Stock price prediction
* RMSE evaluation
* Visualization of actual vs predicted prices

RESULTS

The model successfully learned patterns from historical stock data and generated predictions for future closing prices.

OUTPUT

The model predicts future stock prices based on historical trends.
